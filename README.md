## 🛠 Setup Instructions

- Ubuntu EC2 instance launched on AWS
- Opened ports: 80 (HTTP) and 443 (HTTPS)

### ✅ App Deployment
- Node.js app serves landing page from `/public/index.html`
- Nginx configured as a reverse proxy to Node.js
